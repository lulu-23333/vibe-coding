学习复盘：我的AI编程心得

💭 AI编程心得

从"不会代码"到"用AI写代码"

作为一个完全不会写代码的产品经理，通过这次训练营，我真正体验了"Vibe Coding"的魅力：

  • 该踩的坑都踩了一遍 - 从环境配置到API调用，每个环节都遇到过问题
  • AI是我的编程搭档 - Claude Code帮我生成合同脚本、调试代码、解释错误
  • Prompt调优是关键 - 同样的需求，不同的Prompt，效果天差地别
  • 小步快跑，快速迭代 - 不要一次做太多，先让基本功能跑起来

🗺️ 给新手的学习路径建议

1

从具体痛点出发

不要为了学技术而学技术，先找到自己真正想解决的问题

2

设计最小可行产品（MVP）

想清楚核心功能是什么，先做最简单的版本

3

选择合适的工具

Claude Code + Vercel 是很好的组合，AI辅助 + 快速部署

4

分阶段实现

不要一次做完，先实现核心功能，再逐步优化
